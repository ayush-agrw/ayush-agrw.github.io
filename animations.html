Spiderverse Style Scene

Modeling, texturing, animation and color grading done in Blender.

For the custom effects (halftone, chromatic aberration, vignette, gradients, noise) I used a machine learning based stylization tool which I programmed myself from scratch. This tool aims to immensely simplify and automate the stylization process, making it streamlined enough to be used in larger projects like animated films / short movies.
